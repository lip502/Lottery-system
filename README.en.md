# 基于Javaweb抽奖管理系统

#### Description
基于springboot企业抽奖管理系统平台，springboot项目。

管理员：抽奖内容管理，用户管理，抽奖记录查询，个人中心，菜单管理，抽奖页面。

用户：查询中将记录，个人中心

主要环境：idea+mysql+mybatis+maven+bootstrap
